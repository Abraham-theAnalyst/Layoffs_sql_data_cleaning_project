# Layoffs_sql_data_cleaning_project
Data cleaning project using SQL (MySQL)

## Project Overview
This project demonstrates data cleaning techniques using SQL.</br>
The dataset contained duplicates, null values, inconsistent formats, and unnecesary columns.

## Tools used
<ul>
  <li color:"blue";>MySQL</li>
  <li>SQL workbench</li>
</ul>


## Cleaning Steps
<ul>
  <li>Removed duplicate records</li>
  <li>Standardized date formats</li>
  <li>Handled null values</li>
  <li>Trimmed Unnecessary spaces</li>
  <li>Removed irrelevant columns</li>
</ul>

## Key SQL concepts used
<ul>
  <li>DELETE</li>
  <li>UPDATE</li>
  <li>ALTER TABLE</li>
  <li>CTEs</li>
  <li>ROW_NUMBER()</li>
  <li>PARTITIONS</li>
</ul>

## Outcome
The dataset was transformed into a clean, analysis-ready format.
