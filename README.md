# Layoffs_sql_data_cleaning_project
Data cleaning project using SQL (MySQL)

## Project Overview
This project demonstrates data cleaning techniques using SQL.
The dataset contained duplicates, null values, inconsistent formats, and unnecesary columns.

## Tools used
. MySQL <br>
. SQL workbench 

## Cleaning Steps
. Removed duplicate records <br>
. Standardized date formats <br>
. Handled null values <br>
. Trimmed Unnecessary spaces <br>
. Removed irrelevant columns 

## Key SQL concepts used
. DELETE <br>
. UPDATE <br>
. ALTER TABLE <br>
. CTEs <br>
. ROW_NUMBER() <br>
. PARTITIONS

## Outcome
The dataset was transformed into a clean, analysis_ready format.
